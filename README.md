# GoMLX Documentation Site

The official documentation website for [GoMLX](https://github.com/gomlx/gomlx) — an accelerated machine learning and math framework for Go. Built with [Hugo](https://gohugo.io) (Extended).

## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/gomlx/gomlx-site
cd gomlx-site

# 2. Install Hugo Extended
# macOS: brew install hugo | Linux: snap install hugo --channel=extended | Win: choco install hugo-extended

# 3. Start the local dev server
make dev
# -> open http://localhost:1313
```
